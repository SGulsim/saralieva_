# Лабораторная работа №1

## Тема

Сериализация объектов в формат JSON, работа с файловыми ресурсами на языке C# и реализация паттерна `IDisposable`.

## Структура проекта

- `src/Lab1` — исходный код консольного приложения.
- `Models/Person.cs` — модель данных с вычисляемыми свойствами и валидацией.
- `Services/PersonSerializer.cs` — класс сериализации и десериализации `Person`.
- `Services/FileResourceManager.cs` — обёртка над файловыми ресурсами с поддержкой `IDisposable`.
- `Program.cs` — демонстрация использования разработанных классов.
- `QuestionsAnswers.md` — ответы на контрольные вопросы.
- `README.md` — данный файл.

## Сборка и запуск

```bash
cd src/Lab1
dotnet build
dotnet run
```
